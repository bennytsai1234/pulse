# 🎵 Pulse v2.2.0

## ✨ Experience Enhancements (體驗增強)

### 🎼 歌詞體驗再升級
- **卡拉OK浮動效果**：為全螢幕歌詞添加動態浮出(Float-up)動畫與發光效果，跟隨音樂節奏起伏。
- **歌詞互動優化**：現在可以直接點擊歌詞區域返回專輯封面，底部控制區也添加了專屬歌詞切換按鈕。
- **即時同步修復**：解決了歌詞不隨音樂進度同步移動的問題，實現精確的逐字/逐行高亮。

### 📋 播放隊列 (Up Next) 改進
- **UI 一致性**：統一了返回按鈕樣式與標題列高度，視覺體驗更協調。
- **正確時間顯示**：修復了隊列中歌曲時間顯示為 "00:00" 的錯誤，現在能正確顯示每首歌曲的時長。

### ⚡ 主畫面批量操作
- **長按選取模式**：在主畫面長按歌曲即可進入批量選擇模式。
- **快速管理**：提供批量刪除和批量加入播放清單功能，整理音樂庫更高效。

# 🎵 Pulse v2.1.0

## ✨ 新功能 (New Features)

### 🎧 交叉淡入淡出播放 (Crossfade Playback)
- **無縫歌曲過渡**：開啟交叉淡入淡出後，歌曲之間將平滑銜接，如同專業 DJ 混音效果。
- **可調整時長**：支援 1-12 秒的淡入淡出時長調整，滿足不同聆聽偏好。
- **多種曲線選擇**：提供線性、指數和 S 曲線三種淡入淡出效果。
- **智慧模式**：
  - 手動跳轉時套用：可選擇是否在手動切歌時也使用交叉淡入淡出。
  - 專輯連續模式：同專輯內的歌曲可選擇禁用交叉淡入淡出，保持原始過渡。
- **效能優化**：採用雙播放器架構，記憶體不足時自動降級為普通切換。

### 🔧 技術改進
- 新增 `VolumeAnimator` 單元測試，確保曲線計算準確性。
- 播放器模組測試配置優化。
