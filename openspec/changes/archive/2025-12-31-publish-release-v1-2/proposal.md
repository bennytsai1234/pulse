# Proposal: Publish Release v1.3.0

## Why
在完成了一系列的 UI 優化（手勢、空狀態、歌詞同步）、錯誤修正與工作區清理後，專案已達到一個新的穩定里程碑。發布 v1.3.0 版本可以正式封裝這些改進，並提供給使用者一個更流暢、更具質感的音樂播放體驗。

## What Changes
1.  **版本升級**：
    *   在 `app/build.gradle.kts` 中將 `versionCode` 從 7 提升至 8。
    *   確認 `versionName` 為 "1.3.0"。
2.  **更新發布日誌**：
    *   更新 `RELEASE_NOTES.md`，詳列 v1.3.0 的主要改進項目。
3.  **封裝與標記**：
    *   執行完整建置以驗證 APK 生成。
    *   建立 Git Tag `v1.3.0`。

## Impact
- **穩定性**：通過正式的版本發布流程，確保所有功能在 Release Build 中皆能正常運作。
- **維護性**：標記版本有助於追蹤 Bug 與未來的回溯。
- **使用者體驗**：使用者將獲得包含手勢精練、歌詞優化與全新空狀態的完整更新。
