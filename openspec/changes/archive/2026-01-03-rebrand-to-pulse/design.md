# Design: rebrand-to-pulse

## 設計概述

本文件詳述「Pulse」品牌的視覺識別設計，包含品牌理念、圖標設計規範和動畫設計規範。

---

## 1. 品牌理念

### 1.1 核心概念
- **Pulse (脈動)**：音樂的本質是節奏的脈動，與心跳共鳴
- **視覺隱喻**：脈搏波形、聲波、節拍

### 1.2 色彩系統

| 名稱 | 色碼 | 用途 |
|------|------|------|
| **Primary Cyan** | `#00F2FF` | 主色調，表現能量 |
| **Secondary Blue** | `#0066FF` | 輔助色，表現深度 |
| **Accent Magenta** | `#FF0080` | 強調色，表現活力 |
| **Dark Background** | `#0A0E1A` | 深色背景 |
| **Light Background** | `#E3F2FD` | 淺色背景 |

### 1.3 品牌名稱規範

| 情境 | 英文 | 中文 |
|------|------|------|
| App 名稱 | **Pulse** | **脈動** |
| 完整名稱 | Pulse Music Player | 脈動音樂播放器 |
| 簡稱 | Pulse | 脈動 |

---

## 2. 圖標設計 (App Icon)

### 2.1 設計概念
採用「脈搏線/波形」作為主視覺，結合圓形播放按鈕，在 Adaptive Icon 框架下呈現。

### 2.2 前景圖 (Foreground) - `ic_launcher_foreground.xml`

```
視覺元素：
┌─────────────────────────────┐
│                             │
│      ╭───────────────╮      │
│     ╱     ╲    ╱ ╲   ╲     │  ← 脈搏波形 (Pulse Wave)
│    ╱       ╲  ╱   ╲   ╲    │
│ ──╱─────────╲╱─────╲───╲── │  ← 基準線
│                             │
│          ▶              │  ← 播放按鈕 (Play)
│                             │
└─────────────────────────────┘
```

**設計規格：**
- **尺寸**：108dp x 108dp (Adaptive Icon 標準)
- **安全區域**：66dp x 66dp (中心)
- **波形**：動態漸層 (Cyan → Blue)
- **播放按鈕**：半透明白色

### 2.3 背景圖 (Background) - `ic_launcher_background.xml`

**設計規格：**
- **漸層方向**：315° (左下到右上)
- **色彩**：深藍到深紫的漸層
- **紋理**：細微的波紋圖案增加質感

### 2.4 視覺對比

| 對比項 | 舊設計 (Gemini) | 新設計 (Pulse) |
|--------|----------------|----------------|
| 主視覺 | 雙螺旋 + 音符 | 脈搏波形 + 播放 |
| 色調 | 青藍漸層 | 青藍 + 洋紅強調 |
| 風格 | 科技感 | 動態/生命力 |

## 3. 啟動動畫設計 (Splash Animation)

### 3.1 動畫概念
模擬心跳脈動的效果，波形從中心向外擴散，伴隨播放按鈕的節奏性縮放。

### 3.2 動畫時序

```
時間軸 (毫秒)
0ms     500ms    1000ms   1500ms   2000ms
├────────┼────────┼────────┼────────┤
│◄══ 波形繪製 ══►│                   │  Phase 1: 波形出現
│        │◄══ 脈動縮放 ══►│          │  Phase 2: 圖標脈動
│                │◄══ 光暈擴散 ══►│   │  Phase 3: 光暈效果
│                      │◄═ 淡出 ═►│   Phase 4: 過渡
```

### 3.3 動畫元素

1. **波形繪製 (Path Animation)**
   - 從左向右逐步繪製脈搏波形
   - 使用 `trimPathStart` 和 `trimPathEnd` 動畫

2. **脈動縮放 (Scale Animation)**
   - 中心圖標以 1.0 → 1.1 → 1.0 的節奏縮放
   - 模擬心跳節奏 (~70 BPM)

3. **光暈效果 (Glow Animation)**
   - 半透明圓形從圖標向外擴散
   - Alpha 從 0.5 → 0 漸變

### 3.4 啟動背景 - `splash_background.xml`

**設計規格：**
- **背景漸層**：深色調 (#0A0E1A → #1A1625)
- **裝飾元素**：淡化的波形線條作為背景紋理
- **中心圖標**：Pulse 品牌圖標

## 4. 技術實現考量

### 4.1 Animated Vector Drawable (AVD)
- 使用 `<animated-vector>` 標籤
- 動畫屬性：`rotation`, `scaleX/Y`, `alpha`, `trimPathEnd`
- 插值器：`fast_out_slow_in` 確保流暢感

### 4.2 相容性
- **最低 API**：21 (Android 5.0)
- **Splash Screen API**：使用 androidx.core.splashscreen 相容性函式庫
- **Adaptive Icon**：API 26+ 原生支援，低版本使用 legacy icon

### 4.3 效能考量
- 控制動畫時長在 2 秒以內
- 避免過多同時進行的動畫
- 使用硬體加速的屬性動畫

## 5. 決策記錄

| 決策點 | 選項 | 決定 | 原因 |
|--------|------|------|------|
| 中文名稱 | 脈動 / 心跳 / 節拍 | 脈動 | 與 Pulse 直譯最貼近 |
| 主色調 | 保留青藍 / 全新色系 | 保留青藍 | 維持使用者熟悉度 |
| 圖標風格 | 扁平 / 漸層 / 3D | 漸層 | 現代感與 Premium 質感 |
| 動畫長度 | 1s / 1.5s / 2s | 1.5s | 平衡視覺效果與等待時間 |
