## MODIFIED Requirements

### Requirement: Home Screen Brand Display
The home screen header area SHALL display the correct brand name.

#### Scenario: Home screen title display
- **WHEN** user enters the app home screen
- **THEN** header area SHALL display "Pulse" brand name
- **AND** SHALL use app theme font and colors

### Requirement: Permission Screen Brand Display
The permission request explanation SHALL use the correct brand name.

#### Scenario: Permission explanation display
- **WHEN** user opens app for the first time without granted permissions
- **THEN** explanation text SHALL reference "Pulse" brand name
- **AND** SHALL clearly explain permission usage
